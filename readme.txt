A custom plugin which shows available ski resorts details from fnugg.no API

== Description ==

Fnugg Resorts is a custom WP block which shows weather data for ski resosts and Alpine resorts throughout Norway.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fnugg-resorts` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= How to use the Fnugg Block? =

Once the plugin has been activated. Go to your page, click the [+] icon to add a block and search for `Fnugg Resorts`. Once the block has been added, search for a ski resorts in the block setting sidebar and select the one you would like to show.
